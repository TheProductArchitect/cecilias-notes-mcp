# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and focus on what's best for
the project and its users. Harassment, personal attacks, and
discriminatory language are not tolerated.

## Reporting

If you experience or witness behaviour that violates this code, please
email the maintainer at **nvg1996@gmail.com**. Reports are handled
confidentially.

The full text of the Contributor Covenant is at the link above.
